Memory


Hard Drive

The hard drive is used as a more permanent storage area.

All applications, Operating System, Software, etc... will be stored in the hard drive

There are currently three types of hard drives

  1. SSD (Solid State Drive)
  2. HDD (Hard Disk Drive)
  3. m.2 SSD

The fastest is currently m.2 SSDs, and the slowest is the HDD

This also means that the HDD is the cheapest while the m.2 SSDs is the most expensive

Hard drives speeds are measured by Read and Write speeds


RAM

DRAM or Dynamic Random Access Memory is also a unit that stores memory

The difference between RAM and Hard Drives is that RAM will be a temporary place to hold memory

This is a place to "cache" information or temporarily store information so that your computer

can grab information from memory quicker

The many applications will be using RAM to store all of the information needed to run

The speed is measured in GHz
